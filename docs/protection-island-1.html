The information below is taken from the following source: Newcomb, HR. 1940. Ring-necked pheasant studies on Protection Island in the Strait of Juan de Fuca, Washington. MS thesis. Oregon State University.

  1. Pheasant chicks are born during the summer.
  2. In May 1937, 10 pheasants were introduced to the island. Before the next breeding season there were 35.
  3. November 10, 1938 a census estimated 110 pheasants.
  4. October 13, 1939 a census estimated 400 pheasants.

  2. All the examples of making a graph in Chapter 11 of Quantitative skills for biology have lists of numbers as the values that are plotted on the axes. We would like to make a graph of the data listed in b.-d., however, we will face a problem since the x-axis values are dates, rather than numbers. We will consider two possible solutions:

  1. Use a built-in R function, so that dates can be plotted on the x-axis; and

  2. Convert the dates to number of days since a reference date. Now the dates are numbers and these values can be plotted on the x-axis.

In this question, we will proceed with option i. The function we will use is as.Date(). You can learn how to use this function using an internet search or by typing the following into your Console:

?as.Date

These files can be difficult to understand. A good way to proceed is to experiment with the function in the Console. Try these:

as.Date(2012-01-31, format = %Y-%m-%d)
as.Date("2012-01-31", format = "%Y-%m-%d")

Note that only the second command is error-free. The first command fails needs to contain "" because the date argument for the as.Date() function must be a character string (see ?character). It is also possible to omit the format argument and just code as.Date("2012-01-31"). The help file notes that when the format argument is not specified, the help file states that formats will be tried one by one and an error will be returned if none work. It is advisable to specify the format, as allowing the function to infer the format could introduce errors.

Section 6.9 of Quantitative skills for biology describes how to make a vector. We need to make a vector of the dates so that we can make our plot. For example,

x <- as.Date(c("2012-01-31", "2012-03-05", "2013-01-11"), format = "%Y-%m-%d")
